# Milestone Roadmap

Checked = shippable and committed. Each milestone leaves the site in a
working state.

- [x] **Milestone 0 — Foundation**
      Astro + Tailwind scaffolded, project docs created, Git initialized,
      first commit made, dev server verified locally.
- [ ] **Milestone 1 — Typography & fonts**
      Self-host Space Grotesk, establish the real type scale in
      DESIGN_SYSTEM.md, replace ad-hoc Tailwind classes on the homepage
      with it.
- [ ] **Milestone 2 — Site skeleton**
      Real routes for Home / Work / Experiments / Studio / Contact (static
      placeholder content), shared navigation, confirm Astro's file-based
      routing end to end.
- [ ] **Milestone 3 — Hero: stillness state**
      Build the empty, unmoving hero (Anyrvaan / descriptor / tagline) with
      zero motion — the "before" state the rest of the philosophy depends
      on.
- [ ] **Milestone 4 — Hero: motion born from stillness**
      GSAP-driven dissolve into particles, cursor-as-gravity interaction,
      reform on scroll-up. The signature interaction of the site.
- [ ] **Milestone 5 — Smooth scroll (Lenis) + reduced-motion fallback**
- [ ] **Milestone 6 — Case study template**
      Context → Thinking → Exploration → Iterations → Lessons → Outcome
      structure, built as a reusable layout/content collection.
- [ ] **Milestone 7 — Deployment**
      GitHub repo, Cloudflare Pages connection, build command, preview
      deployments, custom domain + DNS + SSL.
- [ ] **Milestone 8 — Polish pass**
      Accessibility audit, SEO basics, performance check, responsive pass.

Later / not yet scheduled: Journal, Research, Music, Tools sections;
`localStorage`-based gentle adaptation (see PROJECT_CONTEXT.md).
