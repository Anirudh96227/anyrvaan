The anxious version of the AI conversation asks "will designers be replaced?" — as if it's one group with one fate. I don't think it is. Two designers can use the exact same tools every day and end up in completely different positions, because the question was never really whether you use AI. It's whether the value you add is the value the tool already provides. One of those designers gets amplified by the tool. The other is quietly competing with it, and probably losing.

## The line is what you contribute, not what you use

Using AI is table stakes now — nearly everyone will, the way nearly everyone uses a search engine. That's not a differentiator, and it never will be. What differentiates is what you bring that the tool doesn't: judgment, taste, problem definition, the ability to know which output is right and say why. If your contribution is generation itself — cranking out competent variations — you've volunteered for the one job the machine already does best.

<div class="table-wrap">

| Replaceable by the tool | Amplified by the tool |
|---|---|
| Contributes generation — makes variations | Contributes judgment — chooses the right one |
| "I can produce a lot, fast" | "I know which of these is right, and why" |
| Executes a given brief | Defines the brief worth executing |
| Competes with the model on its turf | Does what the model can't |
| Value = output volume | Value = discernment |

</div>

## How to land on the right side

The move, as far as I can tell, is to make your contribution the thing the tool actually lacks. Get demonstrably good at the judgment calls — spotting the generic, articulating why one option beats twenty others, defining problems worth solving in the first place. (I wrote about the core skill in [AI Made Creation Cheap. Taste Got Expensive.](/blog/ai-made-creation-cheap-taste-got-expensive/), and the discipline of choosing among outputs in [Twenty Images, One Keeper](/blog/twenty-images-one-keeper/).) None of this is about rejecting the tools. The amplified designer uses them constantly. It's about making sure the part *you* own is the part that doesn't get cheaper as the models improve.

The reason this matters urgently right now is that the gap is widening fast. Every model improvement makes raw generation cheaper, which makes "I produce a lot" worth less and "I know what's worth producing" worth more. You don't get to opt out of that shift, unfortunately. You only get to choose which side of it your skills end up on.
